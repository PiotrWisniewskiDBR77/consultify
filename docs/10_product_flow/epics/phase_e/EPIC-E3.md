# EPIC-E3: Snapshot as Discussion Starter

| Field | Value |
|-------|-------|
| **Epic ID** | EPIC-E3 |
| **Phase** | E — Guided First Value |
| **User State** | `FIRST_VALUE` |
| **Priority** | HIGH |

---

## Intent

Snapshot **opens dialogue**, not closes it.

---

## System Requirements

### MUST
- [ ] Present snapshot as **starting point**
- [ ] Encourage **revisiting and updating**
- [ ] Frame as living document

### MUST NOT
- [ ] Present snapshot as final
- [ ] Create lock-in pressure
- [ ] Discourage changes

---

## Acceptance Criteria

| Criteria | Pass Condition |
|----------|----------------|
| Snapshot feels **revisitable** | ✅ |
| User sees it as **conversation opener** | ✅ |
