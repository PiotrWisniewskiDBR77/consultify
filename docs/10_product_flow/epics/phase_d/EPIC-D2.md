## Intent

**Speed is the enemy of ownership.**

---

## System Requirements

### MUST
- [ ] Avoid **defaults**
- [ ] Force **conscious input**
- [ ] Make each step meaningful

### MUST NOT
- [ ] Pre-fill organization details
- [ ] Skip confirmation steps
- [ ] Optimize for time-to-completion

## Acceptance Criteria

| Criteria | Pass Condition |
|----------|----------------|
| Setup takes **longer than SaaS norm** | ✅ |
| Each input feels **intentional** | ✅ |
