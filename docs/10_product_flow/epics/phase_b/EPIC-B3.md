# EPIC-B3: Exit by Choice, Not Design

| Field | Value |
|-------|-------|
| **Epic ID** | EPIC-B3 |
| **Phase** | B — Demo Session |
| **User State** | `DEMO_SESSION` |
| **Priority** | HIGH |

---

## Intent

Leaving the demo **without action must feel correct**.

---

## System Requirements

### MUST
- [ ] Provide a **single, calm exit CTA**
- [ ] Store **no intent data**

### MUST NOT
- [ ] Track exit intent
- [ ] Show retention popups

---

## Acceptance Criteria

| Criteria | Pass Condition |
|----------|----------------|
| User can **return later** with no loss or pressure | ✅ |
| Exit feels **respected**, not abandoned | ✅ |
