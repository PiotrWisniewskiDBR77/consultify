## Core Principles

### Trust Progression
```
ANON → DEMO → TRIAL → ORG → VALUE → TEAM → ECOSYSTEM
  ↓       ↓       ↓       ↓       ↓       ↓
 None   Read   Limited  Memory  Dialog  Growth
```

### AI Role Evolution
| Phase | AI Mode | Behavior |
|-------|---------|----------|
| A | **ABSENT** | No AI presence |
| B | **NARRATOR** | Explains, doesn't interact |
| C | **GUIDE** | Explains limits, sets expectations |
| D | **MEMORY** | Activates persistent context |
| E | **PARTNER** | Thinking partner, not solver |
| F | **FACILITATOR** | Surfaces disagreement, synthesizes |
