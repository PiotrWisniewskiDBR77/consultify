# EPIC-G1: Referrer as Quality Filter

| Field | Value |
|-------|-------|
| **Epic ID** | EPIC-G1 |
| **Phase** | G — Ecosystem |
| **User State** | `ECOSYSTEM` |
| **Priority** | MEDIUM |

---

## Intent

Referrers (Affiliates) act as the **first layer of trust** for new users.

---

## System Requirements

### MUST
- [ ] Allow users in Phase G to generate unique referral codes.
- [ ] Track conversions (Trial Entry) attributed to specific referrers.
- [ ] Provide a dashboard for referrers to see their impact.

### MUST NOT
- [ ] Use financial incentives (maintain value-driven growth).
- [ ] Allow spamming or bulk code generation.

---

## Acceptance Criteria

| Criteria | Pass Condition |
|----------|----------------|
| Referrer feels **ownership** of the method's growth | ✅ |
| New users feel **safe** entering via a trusted source | ✅ |
