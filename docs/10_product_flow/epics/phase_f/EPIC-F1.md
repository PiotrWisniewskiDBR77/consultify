# EPIC-F1: Multiple Perspectives as Necessity

| Field | Value |
|-------|-------|
| **Epic ID** | EPIC-F1 |
| **Phase** | F — Team Expansion |
| **User State** | `TEAM` |
| **Priority** | CRITICAL |

---

## Intent

**Growth is driven by complexity.**

System naturally requests more perspectives when decisions require them.

---

## System Requirements

### MUST
- [ ] Explicitly **request more perspectives**
- [ ] Frame expansion as **decision quality**
- [ ] Make invitation feel natural

### MUST NOT
- [ ] Push growth without reason
- [ ] Gamify invitations
- [ ] Create pressure

---

## Trigger

```
SYSTEM: "This decision requires more than one perspective."
ACTION: Invite 2–3 team members
```

---

## Acceptance Criteria

| Criteria | Pass Condition |
|----------|----------------|
| Invitation feels **necessary**, not promotional | ✅ |
| Team expansion serves **decision quality** | ✅ |
